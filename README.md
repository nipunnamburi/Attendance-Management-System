# 🕒 Attendance Management System

A beginner-friendly **web-based Attendance Management System** built using **Java (Spring Boot)**, following the **MVC architecture**.  
This project allows admins to manage student or employee attendance while users can view their individual records.  
It demonstrates secure role-based access, CRUD operations, and database integration with MySQL.

---

## 🚀 Features

- **Admin Module** – Add, update, and delete attendance records.  
- **User Module** – View attendance history and personal attendance data.  
- **Role-Based Access** – Separate privileges for admin, student, and faculty.  
- **Data Persistence** – Attendance data stored and retrieved from MySQL database.  
- **Authentication** – Secure login and role management using Spring Security.

---

## 🧩 Tech Stack

| Layer | Technology |
|-------|-------------|
| Backend | Java, Spring Boot |
| Frontend | Thymeleaf (HTML, CSS, Bootstrap) |
| Database | MySQL |
| Security | Spring Security |
| ORM | Hibernate (JPA) |
| Architecture | MVC (Model-View-Controller) |

---

## 🧠 Learning Outcomes

- Understand **MVC architecture** with Spring Boot.  
- Perform **CRUD operations** with MySQL and Hibernate.  
- Implement **role-based authentication** and secure web access.  
- Integrate frontend templates with Thymeleaf.
